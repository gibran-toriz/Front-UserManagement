# CryptoPayroll application 

## Overview

This repository contains a Node.js project of a company site, allowing user to register and login.

## Prerequisites

- Node.js 18

## Dependencies

- axios 1.6.2
- cookie-parser 1.4.4
- debug 2.6.9
- dotenv 16.3.1
- ejs 2.6.1
- express 4.16.1
- http-errors 1.6.3
- morgan 1.9.1
- web3 4.3.0

## Usage

1. **Build**
   ```bash
   npm install
   ```
2. **Run locally**
   ```bash
   npm start
   ```
